🧪 Design of Experiments (DOE) Integration

What is Design of Experiments (DOE)?

DOE is a systematic approach to planning experiments that helps you understand how design parameters affect cyclone performance with minimal testing. Instead of testing randomly, DOE strategically selects parameter combinations to maximize information gain.

🎯 DOE Methods Explained:

  • Full Factorial: Tests all possible combinations (2^n experiments). Best for 2-3 parameters.
  • Fractional Factorial: Tests selected combinations (2^(n-1)). Efficient for screening many parameters.
  • Central Composite: Adds center and axial points to factorial. Ideal for response surface modeling.
  • Latin Hypercube: Space-filling design. Good for exploring large parameter spaces efficiently.

📊 Key Terms:

  • Parameters (Factors): Design variables you want to study (e.g., diameter, height)
  • Responses: Performance measures you want to optimize (e.g., efficiency, pressure drop)
  • Levels: Different values each parameter takes (e.g., low/high)
  • Runs: Individual experiments with specific parameter combinations
  • Center Points: Experiments at middle values for error estimation

🚀 How to Use This DOE Tool:

  1. Select Parameters: Choose 2-4 cyclone dimensions to study
  2. Set Ranges: Define min/max values for each parameter
  3. Choose DOE Method: Start with "Central Composite" for most cases
  4. Generate Matrix: Creates optimal experiment plan
  5. Run Experiments: Automatically calculates all combinations
  6. Analyze Results: Use built-in charts and statistics to find optimal design
